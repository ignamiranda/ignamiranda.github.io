# Battle Map Generator

Generate fantasy TTRPG battle maps with a single click. Uses a Wave Function Collapse algorithm to place ~50 tiles across 6 biomes onto a 20&times;20 grid.

- **🔄** — Generate a new map
- **📋** — Copy the map as Unicode emoji text
- **💾** — Download the map as a PNG image
- **🖥️** — View fullscreen
